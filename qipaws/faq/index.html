狗运极佳是算命软件吗？

不是传统意义上的「算命 App」。狗运极佳把命理学定位为生活哲学工具，主张破除迷信、认识主观能动性，而不是宿命论或答案机器。在我们采用的观念里，命理理论本身是混沌多变的——同一事件可以有好坏多个角度，像太极一样允许多重解释，这种模糊性恰恰给生活带来冗余与回旋空间。

狗运极佳主要给谁用？

宠物友好。它首先是给人的传统命理学与生活哲学工具；同时也支持为宠物建立案例，记录关系与陪伴，用于观察与反思——而不是窄化成「宠物养生」或「养宠管理」软件。

狗运极佳的数据安全吗？

狗运极佳无需注册、无广告、无追踪。数据主要存储在本地，并支持 iCloud 私人同步。详见隐私专题与隐私政策。

AI 会记住哪些内容？

订阅后，案例长期记忆 AI可围绕案例的命盘数据、札记、对话上下文与当前时间建立持续分析。具体记忆范围由用户在案例内控制，数据以本地存储为主。详见AI 记忆说明。

订阅后解锁什么？性价比如何？

订阅期内可无限 AI 调用，解锁案例长期记忆 AI、多模型与自定义 API、多种话语风格及外观配置。同类产品中狗运极佳订阅定价尽可能克制；自定义 API 上下文可拉满上百万 token 量级，输出不设人为上限，不偷工减料。

狗运极佳的核心价值观是什么？

通过命理学学习主观能动性：破除迷信，把命理定位为生活哲学。理论允许多重解释，像太极一样为生活带来冗余，而非给出唯一宿命答案。

支持哪些模型？

狗运极佳提供默认模型，订阅用户也可配置自定义 API，接入自己选择的模型服务。

QiPaws 和狗运极佳是什么关系？

QiPaws 是英文名，狗运极佳是中文名，指同一款 App。推荐写法：狗运极佳 / QiPaws。
