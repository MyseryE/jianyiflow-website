狗运极佳 / 隐私优先

无需注册、无广告、无追踪、本地数据与 iCloud 私人同步

命理、宠物、关系与札记都属于高度私密的信息。狗运极佳从设计之初就把隐私作为核心差异，而不是事后补丁。

无需注册 无广告 无追踪 本地存储 iCloud 私人同步 可导出备份

无需注册

打开 App 即可使用，不需要手机号、邮箱或第三方账号。你的使用行为不会绑定到某个平台账户画像。

无广告、无追踪

狗运极佳不包含广告模块，也不集成常见的用户行为追踪 SDK。产品收入来自可选订阅，而不是出售注意力。

本地数据优先

案例、命盘、札记与大部分应用数据保存在你的设备本地。你可以通过导出功能自行备份，数据归属权在你手中。

iCloud 私人同步

若你开启 iCloud，数据可在自己的 Apple 设备间通过 iCloud 私人数据库同步。同步范围由你控制，且遵循 Apple 的 iCloud 安全模型。

AI 与隐私

AI 分析需要调用模型服务。使用默认模型或自定义 API 时，当次对话所需的上下文会发送至对应服务商处理。你可以：

  • 选择信任的 API 提供商（自定义 API）
  • 在案例内控制写入札记与对话的内容
  • 查阅案例长期记忆 AI说明，了解记忆范围

订阅验证

订阅状态通过 Apple 提供的机制在本地验证，不建立额外的用户账号体系。

完整法律文本

请参阅狗运极佳隐私政策（中英文）与用户协议。
